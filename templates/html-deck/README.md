# HTML Slide Template

Copy this folder to `weeks/YYYY-MM-DD-topic-slug/slides/` and edit `index.html`.

Preview through a local server:

```bash
python3 -m http.server 8765
```

Then open `http://127.0.0.1:8765/weeks/YYYY-MM-DD-topic-slug/slides/`.

Controls:

- Right / Space: next step or slide
- Left: previous slide
- `G`: toggle overview grid
- `D`: toggle demo panel on supported slides
