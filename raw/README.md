# Raw Material

This folder stores immutable captures and reference material.

- `clips/`: article clips, copied markdown, PDFs, transcripts, and week-specific source packets.
- `assets/`: downloaded images and media.
- `external-repos/`: pinned snapshots of referenced repositories.
- `references/`: durable raw reference documents not tied to one week.

Do not rewrite raw captures. Add new captures or annotations elsewhere.

For a one-week reading, prefer `raw/clips/YYYY-MM-DD-topic-slug/`. For a durable source likely to be reused across weeks, `raw/references/` is fine.
