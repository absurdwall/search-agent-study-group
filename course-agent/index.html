Project boundary

The two repositories have separate responsibilities now and a clear integration direction for later deployment.

Website repo
Owns the published course pages, glossary, and weekly materials.
Learner lab
Owns notebooks, the Agent package, tests, and assignments.
Later
The website will publish course-content data for an Agent backend, and the website chat widget will call that backend over HTTPS.

No content API, backend, or chat widget is part of the first version.
