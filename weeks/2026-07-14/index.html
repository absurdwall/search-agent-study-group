Week 1

Agentic AI systems and a complete ADK run

July 14, 2026 · Search Agent Study Group

Materials

  • Google ADK shop-agent notebook The instructor demo used for the prompt-to-output walkthrough.
  • Search Agent Lab The clean learner repository for setup checks and later assignments.
  • Google ADK: Get started with Python Public reference for setting up and running an ADK agent.
  • Search Agent Group ADK playbook Our living, tested setup guide for the study group.

Goals

  1. Build a high-level map of an agentic AI system and why each component matters. This will serve as a roadmap for later weeks.
  2. Walk through one complete agent run in Google ADK, from the user prompt to the final response.

Coverage plan

  1. Map the core system components: model and instructions, tools, orchestration, state and memory, approvals and guardrails, and evaluation.
  2. Use the shop-agent notebook to follow an initial user prompt, model requests, tool calls, agent delegation, and the final response.
  3. Connect the walkthrough back to the system map and identify questions for future weeks.
