Optional assignment

This public learner-lab exercise is separate from the internal Sparky walkthrough in the Week 2 session.

What you will practice

  • Separate Agent instructions from registered tool capability.
  • Read the function-tool contract generated from a Python function.
  • Use ADK Web events and traces to find and verify a problem.

Complete the assignment

Before you run the notebook

Pull the current week-02 state of your local Search Agent Lab checkout. Then activate its .venv and run python -m pip install --upgrade pip, followed by python -m pip install --require-hashes -r requirements.lock. The lock now includes google-adk[eval] for the optional post-checkpoint eval.

  1. Run the baseline, same-instruction tools=[], and glossary-backed Agents.
  2. Compare all three runs in ADK Web.
  3. Diagnose the missing tool and the weak tool contract.
  4. Repair the function-tool contract.
  5. Run the repaired Agent and verify the result.
  6. Complete the Week 2 checkpoint.
