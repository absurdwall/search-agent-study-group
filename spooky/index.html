Ask Spooky

Ask one question about the published study-group glossary. Spooky answers one turn at a time and links to the supporting material it found.

Press ⌘ + Enter or Ctrl + Enter to submit.

Project boundary

The two repositories have separate responsibilities in the live Spooky experience.

Website repo
Owns the published pages, glossary, weekly materials, and browser interface.
Learner lab
Owns the Agent and API implementation, tests, and learner notebooks.
Connection
The public site calls the deployed Spooky API over HTTPS.

The public interface answers from published study-group material only.
