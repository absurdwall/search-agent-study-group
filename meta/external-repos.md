# External Repos

Use this file to track repos referenced or studied by the group.

Repo snapshots belong under:

```text
raw/external-repos/<owner>__<repo>/
```

Each snapshot should include a `.source.json`:

```json
{
  "repo": "https://github.com/owner/repo",
  "commit": "full_commit_sha",
  "captured_at": "YYYY-MM-DD",
  "reason": "Why this repo matters for the study group"
}
```
